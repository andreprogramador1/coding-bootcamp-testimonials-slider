# Desafio frontend-mentor

Este projeto foi um estudo de html5, css e javascript com base em um layout tirado do site [frontendmentor.io](https://www.frontendmentor.io/challenges/coding-bootcamp-testimonials-slider-4FNyLA8JL).

## Pré-requisitos

- Navegador (Chrome, FireFox, Edge, etc...)

## Executar

1. Clonar repositório em algum diretório.
2. Abrir o arquivo index.html.
3. Clicar nos botões de voltar e avançar.
